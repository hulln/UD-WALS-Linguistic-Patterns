# Project 1 — data inventory

See the repo-wide index in [`../../DATA.md`](../../DATA.md). Tracked = pushed to
git; gitignored = kept locally only.

## Source corpora — `data/src/` (gitignored)

Public Universal Dependencies Slovenian treebanks (re-downloadable):

| File | What | Source |
| --- | --- | --- |
| `sl_ssj-ud.conllu` | **Written** Slovenian (UD Slovenian-SSJ), raw | https://github.com/UniversalDependencies/UD_Slovenian-SSJ |
| `sl_sst-ud.conllu` | **Spoken** Slovenian (UD Slovenian-SST) | https://github.com/UniversalDependencies/UD_Slovenian-SST |
| `fixed_sl_ssj-ud.conllu` | SSJ after ID fix | regenerated by `scripts/2_fix_and_validate_conllu.py` |
| `cleaned_sl_ssj-ud.conllu` | SSJ, punctuation removed (the file STARK extracted from) | regenerated by `scripts/3_remove_punct_conllu.py` |

Preprocessing chain: `sl_ssj-ud` → (script 2) → `fixed_` → (script 3) → `cleaned_`.
The `.conllu` are gitignored; to recreate them, download the UD treebanks and run
scripts 2 then 3.
