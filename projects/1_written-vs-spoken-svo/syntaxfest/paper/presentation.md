# Hello everyone, my name is Nives Hüll, and my poster explores word order variation across speech and writing.

GAP		Typological databases like WALS or Grambank usually describe languages as if they had one dominant word order, like Subject–Verb–Object. But these descriptions are almost always based on written sources. That leaves out a huge part of how we actually use language: spontaneous speech. And that’s the gap this project addresses.

PROCESS	We compared spoken and written corpora for five Indo-European languages: English, French, Norwegian Nynorsk, Slovenian, and Spanish, using Universal Dependencies treebanks, looking at how often each of the six possible word orders appears.

RESULT	Across all five languages, the pattern is the same: spoken language is more flexible than writing. The default SVO order drops, and alternatives like SOV, OSV, and OVS rise. English stays the most rigid, while Slovenian and Spanish show the most variation.

CAUSE		What drives this flexibility? Several factors: morphological richness, which allows roles to be marked outside of word order; prosody guides the listener, and information structure, where for example topic–focus organization reshapes the sequence in real time.

TAKEAWAY	So the main takeaway is: if we only look at writing, we may miss how languages actually work. Spoken corpora reveal patterns that challenge neat textbook descriptions — which matters not just for typology, but also for fields like NLP.

Thank you.
